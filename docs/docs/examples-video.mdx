The easiest way to start monetizing your videos is to copy the ```examples/audioAndVideo``` as your base project. Here we go through some of the components being used, so you can modify them to better suit your application.

## Startup

```
npm run streamExample
```

## Frontend

Import the web component from HTML:

```HTML
<script src="packages/wmm-web-components/videoComponent.js"
        type="module">
</script>
```

Or from JS:

```
import './packages/wmm-web-components/videoComponent.js'
```


And insert it into a HTML page:

```
<wmm-video src="[video source url]"
           paymentURL="[recipe service url with payment pointer]">
</wmm-video>
```
